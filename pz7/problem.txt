Задание:
    1. Изучить алгоритмы бинарного поиска из библиотек С/С++.

    2. Реализовать метод поиска в упорядоченном файле, представленном в виде массива: 
    Метод поиска выбирается следующим образом (N – номер по журналу):
        1) если N % 4 = 0  -   фибоначчиев поиск,
        2) если N % 4 = 1  -   интерполяционный поиск,
        3) если N % 4 = 2  -    индексно-последовательный поиск
        4) если N % 4 = 3  -   бинарный поиск

    3. Реализацию алгоритма оформить в виде шаблона функций, которой в качестве параметров 
    передаются два указателя (первый идентифицирует начало диапазона поиска, а второй указывает 
    на ячейку памяти, следующую за последним элементом из диапазона) и искомое значение value. 
    Функция возвращает значение true, если значение value имеется в диапазоне [first  last), 
    и false в противном случае (при желании можно возвращать указатель на найденный элемент массива).

    4. Создать тестирующий проект, в котором протестировать шаблон функций для int и double.
